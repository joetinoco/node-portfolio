Wikiwhat

While experimenting with Wikipedia's API, I built WikiWhat, a dictionary that cross-references articles in multiple languages to find translations. It is very useful for terms or expressions that you can't find in a conventional dictionary, like technical terms or movie names.

Languages used: JavaScript (with jQuery)

APIs used: Wikipedia

Blablabra

In 2001, before Twitter launched "trending topics" for specific cities and countries, I devised a similar algorythm and, using Twitter API and each tweet's geolocated data, launched a site that calculated the trending topics for Brazil. It also included a search function, trending charts and several other functionalities.

Recently, this project was updated into a geolocation oriented, in-browser implementation of the trending topics algorithm. Now, it works anywhere you point your map to, from your neighbourhood to an entire continent.

Languages used: PHP, JavaScript

APIs used: Twitter, Google Maps
